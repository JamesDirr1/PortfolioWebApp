# Version 0.1.0

---

Initial release of the project, including the core API, logging, testing, and documentation.

---

## Added

### Core Application

- Initial ASP.NET Core Web API project
- Layered architecture:
    - API
    - Application
    - Domain
    - Infrastructure
- PostgreSQL integration with EF Core
- Basic Categories endpoint

### Observability

- Logging with Serilog
- Request tracing with `Request-Id`

### Testing

- Unit testing setup with xUnit
- CI pipeline with GitHub Actions

### Documentation

- API documentation
- Architecture documentation
- Setup documentation
- Testing documentation
